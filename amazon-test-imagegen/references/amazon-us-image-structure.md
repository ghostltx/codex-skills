# Amazon US Product Image Structure

## Complete Image Set

Amazon US listing image sets usually contain 7-9 images:

- 1 main hero image
- 6-8 secondary images
- Optional A+ detail-page modules

Default production sizes:

- Main image: `2000 x 2000 px`
- Secondary images: `1600 x 2000 px`
- A+ images: `1464 x 600 px`
- Preferred format: `JPG`

Default file placement:

- Save line-art constraint images beside their matching original product images.
- Save final generated ecommerce images in an `output` folder under the current project directory.

## Main Hero Image

Purpose: improve CTR by creating first-glance attraction.

Requirements:

- Canvas is `2000 x 2000 px`.
- Product is clear and dominant.
- Product occupies about 80% or more of the frame when category-appropriate.
- Background is pure white `#FFFFFF`.
- Output format should prefer `JPG`.
- Premium feeling comes from lighting, material, composition, and realism.

Avoid:

- Large copy
- Logo stacking
- Watermarks
- Icons
- Borders
- Unprovided gifts
- Fake scenes
- Function-heavy poster layouts

Common forms:

- Furniture: white-background product image or subtle lifestyle-white integration
- Outdoor products: white-background product image with structure emphasized
- Small appliances: white-background product with included accessories when accurate

## Secondary Image Template

Default secondary-image canvas: `1600 x 2000 px`.

### Image 2: Lifestyle Scene

Purpose: build user identification and help customers imagine use after purchase.

Show realistic American life contexts such as:

- Family/home scene
- Backyard
- Kitchen
- Office
- Bedroom

Avoid fake model posing. Prefer authentic, believable, natural usage.

### Image 3: Function or Structure

Purpose: explain what the product does.

Show:

- Structure
- Zones or compartments
- Functional points
- Accessories
- Core selling points

Use a minimal, clean layout with breathing room.

### Image 4: Dimensions

Purpose: reduce returns caused by size misunderstanding.

Must include:

- Inch measurements
- Human scale or spatial reference
- Real placement context when helpful

Examples:

- Living room
- Desktop
- Backyard
- Garage

### Image 5: Material or Detail

Purpose: establish quality and premium feeling.

Possible details:

- Wood grain
- Stitching
- Texture
- Waterproof surface
- Screws
- Metal
- Fabric

### Image 6: Multi-Scene Usage

Purpose: communicate multi-use value.

Examples:

- Potting bench: potting bench, BBQ prep, outdoor kitchen, storage station
- Storage rack: kitchen, bathroom, garage, office

### Image 7: Assembly or Packaging

Purpose: answer setup concerns.

Show:

- Assembly process
- Included hardware
- Easy setup
- Packaging

Useful copy:

- Easy to Assemble
- Setup in Minutes
- All Hardware Included

### Image 8: Brand Atmosphere

Purpose: establish brand feeling for mid/high-end products.

Style references:

- Pinterest
- Instagram
- Apple
- Pottery Barn

## A+ Detail Page

A+ should not simply repeat secondary images. It should build brand and lifestyle value.

Default A+ canvas: `1464 x 600 px`.

Common modules:

- Brand banner
- Lifestyle banner
- Function breakdown
- Material details
- Comparison chart
- Brand story

## US Visual Direction

Amazon US customers generally prefer:

- Clean
- Restrained
- Premium
- Realistic
- Authentic
- Emotional
- Lifestyle-oriented
- Minimal
- Cozy
- Natural

The core conversion idea is the ideal life after using the product, not just product parameters.
