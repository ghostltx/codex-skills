# Hive AI Content Check

Use this skill for Hive Moderation AI-generated content detection tasks, especially when the user mentions Hive, Hive Moderation, AI生成检测, 深伪检测, 图片是否AI生成, or API额度.

## Workflow

1. Prefer the official docs and pricing page as the source of truth.
2. For image checks, use the AI-generated image/video detection endpoint.
3. Report the current published free-tier or trial quota if available; do not invent daily limits.
4. If the user asks for integration help, provide a minimal request flow and note authentication requirements.
5. If the quota is unclear, say so explicitly and point to the pricing or docs page.

## Output shape

- What the service does
- Whether it is free / trial / paid
- Published daily limit or request cap
- Relevant API link
